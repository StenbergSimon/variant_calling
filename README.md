variant_calling
===============

Scripts used to run variant calling in an automated way


Dependencies:
------------

* Samtools
* Freebayes
* Bamtools
* Deeptools
* Picard
* GATK
* snpEFF
* BWA
* vcftools
* bgzip
* tabix
* CNV_pipe
